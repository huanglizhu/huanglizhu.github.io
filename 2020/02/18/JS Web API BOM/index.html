JS Web API BOM

二月 18, 2020 JavaScript学习 本文总阅读量次

BOM

  • BOM操作是Browser Object Model（浏览器对象模型）
  • DOM操作是Document Object Model（文件对象模型）
  • BOM相对DOM来说简单许多，基本就是API的操作，了解即可，需要用到的时候再查就行。

相关知识点

  • navigator：浏览器的信息
  • screen：屏幕的信息
  • location：地址的信息（url）
  • history：前移、后退的信息

location的常用属性

  • location.href:整个网站地址(返回完整的URL)
  • location.protocol:返回一个URL协议（比如https:/http:）
  • location.host:域名(返回一个URL的主机名和端口)
  • location.search:返回一个URL的查询部分，即?后的内容
  • location.hash:网址的哈希(返回一个URL的锚部分)，即#后的内容
  • location.pathname:返回的URL路径名

history的API

  • history.back():后退一页，相当于点击“返回”
  • history.forward():前进一页
