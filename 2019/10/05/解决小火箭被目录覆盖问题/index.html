hexo中hipaper主题——解决小火箭被目录覆盖问题

十月 5, 2019 hexo中遇到过的问题与解决方案 本文总阅读量次

出问题页面F12

  • 发现目录覆盖小火箭

找出样式表

  • 通过style.css找到样式表位置在css文件夹里

全局搜索找到toc-fixed

全局搜索真的很重要！一开始在style.styl里面搜索了好久都找不到

  1. 打开E:\Program Files\Hexo\themes\hipaper\source\css
  2. 在css文件夹内空白处右键打开vscode
  3. 点击左边放大镜图标，全局搜索toc-fixed
  4. 发现我们要修改的实际上在E:\Program Files\Hexo\themes\hipaper\source\css\ partial\article.styl里面
  5. 修改为 width: 350px !important;
  6. 发现小火箭不会被覆盖了
