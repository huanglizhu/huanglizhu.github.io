AsyncStorage 异步存储

AsyncStorage是React Native提供的一个异步、未加密、持久的、全局的键值对(key-value)存储模块

方法 描述
setItem(key,value) 通过key-value（键值对） 存储数据
getItem(key) 通过key 读取数据 value
removeItem(key) 删除 key对应的那条数据
clear() 清除全部数据
